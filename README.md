## Overview

Sph√®re is a full-stack weather application that goes beyond basic weather data. It combines real-time meteorological information with Google Gemini AI to deliver personalized insights, mood-based atmospheric visuals, and a conversational weather assistant ‚Äî all wrapped in an iOS-inspired glassmorphic interface.

---

## Features

- **Real-time Weather Data** ‚Äî Temperature, humidity, wind speed, AQI, UV index, pressure, visibility, sunrise & sunset
- **AI-Powered Insights** ‚Äî Contextual tips generated by Gemini 2.5 Flash based on current conditions
- **Weather Chat Assistant** ‚Äî Ask anything about the weather; the AI knows the full week's forecast
- **Mood-Responsive Backgrounds** ‚Äî Animated canvas orbs and gradients that shift based on weather condition and time of day
- **7-Day Forecast** ‚Äî Daily high/low temperatures with condition icons
- **Geolocation** ‚Äî Auto-detects your city on first load
- **Favourite Cities** ‚Äî Pin cities for quick access
- **Recent Searches** ‚Äî Persisted across sessions via localStorage
- **Temperature Toggle** ‚Äî Switch between ¬∞C and ¬∞F
- **Time Format Toggle** ‚Äî Switch between 12h and 24h
- **Dark / Light Mode** ‚Äî System-aware with manual override
- **Glassmorphism UI** ‚Äî iOS-inspired frosted glass cards with smooth animations

## Getting Started

### Prerequisites
- Node.js 18+
- Python 3.10+
- OpenWeatherMap API key (free tier)
- Google Gemini API key (free tier)

---

### Backend Setup

```bash
# Navigate to backend
cd backend

# Create virtual environment
python -m venv venv

# Activate it
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Create .env file
touch .env
```

Add to `backend/.env`:
```
OPENWEATHER_API_KEY=your_openweathermap_key
GEMINI_API_KEY=your_gemini_key
```

```bash
# Start the server
uvicorn main:app --reload
```

Backend runs at `https://sphere-backend-ypzg.onrender.com`

---

### Frontend Setup

```bash
# Navigate to frontend
cd frontend

# Install dependencies
npm install

# Create .env file
touch .env
```

Add to `frontend/.env`:
```
VITE_OPENWEATHER_API_KEY=your_openweathermap_key
```

```bash
# Start the dev server
npm run dev
```

Frontend runs at `https://sphere-pearlalmeida.vercel.app/`

## Key Concepts Used

- **React Hooks** ‚Äî useState, useEffect, useRef for state and side effects
- **Canvas API** ‚Äî requestAnimationFrame loop for smooth orb animations
- **Framer Motion** ‚Äî Page transitions, staggered animations, gesture handling
- **Glassmorphism** ‚Äî backdrop-filter blur with semi-transparent backgrounds
- **Geolocation API** ‚Äî Browser location + reverse geocoding
- **AbortController** ‚Äî Cancels stale API requests on component unmount
- **LocalStorage** ‚Äî Persists user preferences and search history
